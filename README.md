# 🐱 Ultra-Clicker: Max Growth Edition

A high-intensity, clicker-style browser game where speed is survival. Feed your cat, watch it grow, and climb the ranks of the global elite.

## 🚀 What's New in the "Hardcore" Update?

### 🏆 The Elite 7-Rank System
We've recalibrated the ranking logic to be **10x harder**. It’s no longer just about clicking; it’s about mastery. Can you hit the mythical 20 CPS?
* **DIVINE:** 20.0+ CPS
* **ASCENDED:** 16.0+ CPS
* **MYTHIC:** 13.0+ CPS
* **LEGEND:** 10.0+ CPS
* **ELITE:** 8.0+ CPS
* **ADEPT:** 6.0+ CPS
* **NOVICE:** Entry Level

### 🎨 Midnight Neon Aesthetic
* **Glassmorphism UI:** Frosted glass containers with real-time backdrop-blur effects.
* **Dynamic Feedback:** The cat now pulses and shakes when in "Critical Hunger" mode.
* **New Color Palette:** Deep Midnight Charcoal backgrounds with high-contrast Neon difficulty tiles.

### ⚙️ Technical Features
* **Persistent Records:** High scores and ranks are saved to `localStorage`, tracked uniquely across 5 difficulty modes (Easy to Impossible).
* **Precise CPS Logic:** Real-time calculation using `Date.now()` and total click goals for 100% fair speed tracking.
* **Physics-Based FX:** Fish explosion particles with randomized velocity and rotation.

## 🎮 How to Play
1. **Select Difficulty:** Choose your challenge. Higher difficulties increase the hunger drain.
2. **Click to Feed:** Hit the **FEED** button as fast as possible.
3. **Watch for Jackpots:** Random rolls can give you +5 or +10 boosts.
4. **Survive:** If the hunger bar hits zero, it's Game Over.
5. **Rank Up:** Finish the game to see your official Feeder Rank!
